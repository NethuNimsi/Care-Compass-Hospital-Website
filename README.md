# Care-Compass-Hospital-Website
A comprehensive role-based hospital management system designed to streamline healthcare operations and improve patient care coordination. This web application features secure user authentication, role-based access control for doctors, nurses, and administrative staff, and an intuitive interface for managing patient records, appointments, and medical documentation. Built with a modern technology stack including HTML5, CSS3, JavaScript for the frontend and PHP for backend logic, the system ensures data integrity and secure information handling. The platform includes features such as appointment scheduling, patient registration, medical history tracking, and administrative dashboards, providing a complete solution for hospital workflow management.
